# OOP Turn Based Terminal Game

A turn based game with OOP using python with very minimal help from AI (only used to help color texts)

## Characters

### Kiko

A character with mostly supporting skills such as buffing attacks and team heal

### Ampy

A Demonic Dragon with a high damaging skill and a defense reducing debuff
